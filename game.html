Video Game Design Brief

Design a Game based on any one of the following Genres.

  • Idle or Clicker Game
  • Puzzle Game
  • Stealth Game
  • RPG Game
  • Metroidvania Game
  • Hack and Slash Game

Deliverables

A Pitch Presentation with a detailed Game Design Document and a Paper Prototype or storyboard. (Digital prototypes are optional)

The Game Design Document should present your understanding towards the particular genre, MDA, Level Plan and Flow, Scenarios, Learning and Difficulty Curve, Character Theories, Player Study and Experience Factors.
