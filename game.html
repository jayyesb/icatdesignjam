Game Design Brief

Design a Game based on any one of the following Genres.

  • Idle or Clicker Game
  • Puzzle Game
  • Stealth Game
  • RPG Game
  • Metroidvania Game
  • Hack and Slash Game

Deliverables

A Proposal with a detailed Game Design Document and a Paper Prototype or storyboard. (Digital prototypes are optional)

The Game Design Document should contain no less than overview of the Game, MDA - Gameplay, Level Plan and Flow, Scenarios, Learning and Difficulty Curve, Character Theories (if any), Player Study and Experience Factors.
